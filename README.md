> 1. JSX는 단지 HTML과 자바스크립트를 조합한 문법이다.

> 4. 점 연산자를 대신해서 ES6의 구조 분해 할당을 사용할 수 있다.
> 5. React는 다수의 컴포넌트로 이루어져 있으나, 최종적으로는 하나의 컴포넌트만 사용 가능하다.
